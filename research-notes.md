# Reference Direction

## Source Interface Read

- Black and deep wine background with a soft red glow.
- Oversized editorial serif typography in stacked lines.
- Cream section reversal for the formal invitation moment.
- Fixed top navigation with spaced uppercase labels.
- Thin outline buttons and strict rectangular image/card surfaces.
- Invitation narrative: the visitor is entering a private fashion house.

## New Build Direction

- Rebuilt the page as `THE ROUGE HOUSE`, not the previous invitation concept.
- Kept the supplied red heel as the cover object and Open Graph image.
- Removed the old runway sample images and old `FASHION SO / Red Line` wording.
- Created sections for Hero, Official Invitation, Archive, Runway Selection, and Inside the House.
- Used CSS-made house cards instead of reusing previous stock runway imagery.
